# app/ — React application

This directory contains the full React application: the router, screen-level components, and all reusable UI components. It depends on React Router v7, Framer Motion (`motion/react`), Tailwind CSS, and Lucide icons.

## Responsibility

Render the complete game UI — lobby, pre-game room, active game table, and post-game results — and handle client-side navigation between those screens.

## Structure

| Path | Role |
|------|------|
| `App.tsx` | Application root — enables dark mode and mounts the router provider. |
| `routes.tsx` | Declares all four routes: `/` (lobby), `/room/:roomCode`, `/game/:roomCode`, `/result/:roomCode`. |
| `screens/` | One component per full-page view; owns mock game state and local phase transitions. |
| `components/game/` | Reusable game-domain UI: cards, player seats, bidding panel, score display, overlays. |
| `components/effects/` | Purely decorative animated backgrounds used on the lobby screen. |
| `components/figma/` | Utility wrapper (`ImageWithFallback`) generated by Figma Make. |
| `components/transitions/` | Page transition wrapper using Framer Motion. |
| `components/ui/` | Shadcn/ui primitives (accordion, dialog, button, etc.) — not game-specific. |
